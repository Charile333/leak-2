# 域名检索系统开发文档 (Domain Retrieval System)

## 1. 项目概述
本系统是一个高性能的域名泄露凭证检索平台，旨在帮助安全团队快速发现受监控域名下的潜在风险。系统参考了 `leakradar.io` 的核心逻辑，并进行了 UI/UX 的现代升级。

## 2. 核心功能
- **实时域名检索**：支持对目标域名进行全球范围内的泄露库比对。
- **多维度统计报告**：自动对泄露记录进行分类（员工、第三方、客户）并分析密码强度分布。
- **详细记录查询**：支持查看具体的泄露字段，包括邮箱、明文密码、哈希值、泄露源及关联 IP。
- **交互式过滤与排序**：用户可按日期、邮箱或密码强度对结果进行实时排序和筛选。
- **数据导出**：支持将安全报告导出为 PDF 格式。

## 3. 技术架构
### 前端 (Frontend)
- **框架**: React 19 + TypeScript
- **样式**: Tailwind CSS (使用现代紫色/靛蓝配色方案)
- **动画**: Framer Motion
- **图标**: Lucide React
- **状态管理**: React Hooks (useState, useMemo)

### 后端 (Mock API)
- **服务层**: `src/services/mockDataService.ts`
- **设计模式**: 采用异步服务模式，可无缝切换至 Axios/Fetch 调用真实 REST API。

## 4. 视觉设计规范
- **主色调**: 
  - 背景: `#0a0a0c` (深色沉浸式)
  - 品牌色: `#4f46e5` (Indigo)
  - 强调色: `#a855f7` (Purple)
- **卡片设计**: 采用 `glassmorphism` (玻璃拟态) 效果，搭配 `3xl` 大圆角。
- **反馈设计**: 
  - 检索过程中的 Skeleton/Loader 动画。
  - 密码字段的“点击查看”交互逻辑。

## 5. 性能优化
- **结果分页与过滤**: 使用 `useMemo` 对大规模 mock 数据进行前端实时处理，确保 UI 响应零延迟。
- **懒加载**: 搜索结果区域采用 `AnimatePresence` 延迟加载，提升首屏渲染速度。

## 6. 安全建议 (交付说明)
- 生产环境应强制执行 API 请求频率限制 (Rate Limiting)。
- 敏感字段（如明文密码）在传输过程中必须经过严格加密，并在 UI 端默认脱敏显示。
